About

My name is Heath Anderson. I live and work in Chattanooga, Tennessee. I work at Blacksnow Media. We build cool websites and mobile apps.

This website uses the following technologies/resources:

  • Jekyll
  • Generic Blog Plugins by Jose Diaz-Gonzalez
  • Better Web Readability Project
  • Tileables Pack #2 - Paper
  • Facebook Badges
  • Clicky
  • Amazon S3
  • HTML5
  • CSS3
  • Atom
  • Solarized Dark Pygments CSS
Buzzwords are fun!

This site was build by using my 13-inch Macbook Pro. The HTML and CSS was created and edited using TextMate. TextMate was also used to write the blog posts. I took advantage of Adobe Photoshop to do the small amount of image editing that was required. Jekyll generates static HTML when I write a new post or update the site. The site is then uploaded to and hosted on Amazon S3.
