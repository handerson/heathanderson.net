About

My name is Heath Anderson. I live and work in Chattanooga, Tennessee. I work at Blacksnow Media. We build websites and mobile apps.

This website uses the following technologies/resources:

  • Jekyll
  • Generic Blog Plugins by Jose Diaz-Gonzalez
  • Sass
  • Compass
  • Susy
  • Facebook Badges
  • Clicky
  • Amazon S3
  • HTML5
  • CSS3
  • Atom
  • Solarized Pygments CSS
  • Social Media Icons from IconDock
Buzzwords are fun!

This site was build by using my 13-inch Macbook Pro. The HTML and CSS was created using TextMate. TextMate has now been superseded by Sublime Text 2 as my editor of choice. I took advantage of Adobe Photoshop to do the small amount of image editing that was required. Jekyll generates static HTML when I write a new post or update the site. The site is then uploaded to and hosted on Amazon S3.
